ETIQUETA CANVAS

El "canvas" o lienzo en HTML es un componente integrado en HTML5 que posibilita la creación de gráficos de forma dinámica mediante el uso de scripts.

Entre sus capacidades se incluye la generación en tiempo real de gráficos 2D y mapas de bits, así como la animación de estos elementos.

Es un modelo de procedimiento de nivel bajo, ya que actualiza directamente un mapa de bits y no cuenta con una representación gráfica integrada de la escena.
